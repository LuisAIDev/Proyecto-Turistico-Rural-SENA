# CRUD de Usuarios - Evidencia GA7-220501096

Este proyecto es una API REST desarrollada para la formación del SENA. Permite la gestión completa de usuarios (Crear, Leer, Actualizar y Eliminar) conectada a una base de datos PostgreSQL.

## 🚀 Tecnologías utilizadas

- **Node.js**: Entorno de ejecución para JavaScript.
- **Express**: Framework para el servidor web.
- **PostgreSQL**: Sistema de gestión de base de datos relacional.
- **Postman**: Herramienta utilizada para el testeo de los endpoints.

## 🛠️ Instalación y Configuración

1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Configurar las variables de entorno en el archivo `.env` (DB_USER, DB_PASSWORD, etc.).
4. Iniciar el servidor con `npm start` o `node server.js`.
